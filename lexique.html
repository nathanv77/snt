Lexique

- Le contenu non lié : les pages du site ne sont pas reliés entre elles ; il n’y a donc pas de lien hypertexte. Ce sont des pages sans back links.


- Le contenu de script : les robots chargés de la compréhension du script n’arrivent tout sim-plement pas à les/le lire.


- Le format non- indexable : les formats de donnés ne sont pas « lisible » par les moteurs de recherches*. Ce fut le cas des formats PDF/ MICROSOFT OFFICE.


- Le contenue trop volumineux : il y a nombre limite d’octets pour chaque moteur de re-cherche.


- Le contenu privé : la page web est dirigée par son administrateur. Le seul moyen d’accéder à ses pages est de connaitre l’URL de celle-ci ; c’est le web privé.


- Le contenue à accès limite : c’est le web propriétaire ; il faut avoir « un compte » pour pouvoir accéder à certains contenus.


- Le contenue dynamique : en fonction des utilisateurs, les liens changent et le conte-nue des pages également en fonction des paramètres du site ; ce qui empêchent l’indexation.


- Cybercrime : activités criminelles sur internet.
